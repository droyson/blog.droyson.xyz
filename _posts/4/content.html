Javascript is a programming language used in the browser. In past few years javascript has evolved from being used only within browsers to servers and other IoT devices. That's all the history I would be able to provide on JS for this post.

Moving ahead with the post. Javascript can be used to change the styles of an element, listen to clicks, fetch data from backend servers and a lot more. In here we will limit our usage to a few basic methods of manipulating the HTML elements. So let's get to it.

Now that we have a link to Expand the card let's make it functional.

First step would be to create a js file and include it with the html using the <script> tag.

In the js file add an event listener for DOMContentLoaded so that the JS code is loaded only after the HTML is loaded. This acts as a safety and helps encapsulate the JS code within a function instead of polluting the global namespace.

Get references to all the Expand/Collapse links using querySelectorAll and add click listeners to each link.

When the link gets clicked the event listener callback is triggered in which this is a keyword referencing the element that got clicked. Find the parent article to find out if the card is in collapsed or expanded state and update the class of the article accordingly. We also update the text in the link to display the correct action the link will perform i.e. Expand or Collapse.
